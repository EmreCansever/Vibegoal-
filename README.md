⚽ Canlı Düello & Tahmin Web Uygulaması
Bu proje, futbolseverlerin canlı maçlar üzerinden anlık tahminler yürüterek birbirleriyle rekabet ettiği otonom bir tahmin platformudur. Gerçek zamanlı verilerle beslenen odalar ve dinamik puanlama mekanizması içerir.

✨ Özellikler
Gerçek Zamanlı Veri Entegrasyonu: API-Football kullanılarak en güncel canlı maç verileri, skorlar ve dakikalar odalara anlık akar.

Kota Dostu Önbellek: Ücretsiz API kotasını korumak amacıyla 30-45 saniyelik akıllı fetch periyotları ve otonom puanlama motoru.

Dinamik Temalar: Derin Gece, Cehennem Odası ve Retro Tribün gibi modlarla tamamen dinamik UI deneyimi.

Liderlik Tablosu & Rekabet: Odalardaki kullanıcıların puanlarına göre canlı sıralanan "Elazığ Tayfa" stili Leaderboard.

Sol Kayar Menü (Sidebar): Profil düzenleme (Kullanıcı adı, biyografi, avatar), son 25 maçlık sınırlandırılmış geçmiş tahmin listesi ve genel ayarlar paneli.

PWA Desteği: Mobil cihazlara doğrudan kurulabilir (Ana Ekrana Ekle) yerel uygulama yapısı.

🛠️ Teknolojiler
React / Vite

Tailwind CSS

API-Football (Real-time Sports API)

LocalStorage (Client-side Data Management)

🚀 Kurulum ve Çalıştırma
Projeyi bilgisayarınıza indirin ve klasöre girin.

"npm install" komutu ile tüm bağımlılıkları yükleyin.

Ana dizine bir .env dosyası oluşturup "VITE_FOOTBALL_API_KEY=api_anahtarınız" şeklinde anahtarınızı ekleyin.

Projeyi lokalde başlatmak için "npm run dev" komutunu çalıştırın.
